# **App Name**: Code Extractor

## Core Features:

- Code Input: Provide a text area where the user can paste the HTML code.
- Code Splitting: Automatically parse the HTML code and extract the content within the <style> tags into a CSS file and the content within the <script> tags into a JS file.
- File Download - HTML: Provide a download button for the index.html file, which contains the HTML structure without the inline styles and scripts.
- File Download - CSS: Provide a download button for the style.css file, containing the extracted CSS styles.
- File Download - JS: Provide a download button for the script.js file, containing the extracted JavaScript code.

## Style Guidelines:

- Primary color: Soft blue (#A0C4FF) for a calm, code-focused atmosphere.
- Background color: Light gray (#F0F0F0) to reduce eye strain during long coding sessions.
- Accent color: Orange (#FFB347) for buttons and important actions.
- Body and headline font: 'Inter' for a clean and modern look.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use simple, outline-style icons for file types (HTML, CSS, JS) and actions (download).
- Clear, straightforward layout with a large text area for code input and prominent download buttons.
- Subtle hover effects on buttons to provide feedback.
